# Image References

## Brand Reference

The `brand-reference` folder contains screenshots supplied by the user showing The Wizard Nexus website branding, including:

- the official logo treatment;
- serif typography;
- gold and ivory brand colors;
- deep-space visual background;
- restrained, spacious composition.

These screenshots should guide future visual work.

## Generated Concepts

The `generated-concepts` folder contains 18 generated visual iterations from the ideation process.

They are useful for:

- architecture visualization;
- feature hierarchy;
- color experiments;
- wording exploration;
- identifying layout failures and improvements.

They are **not** authoritative brand assets.

Known issues across some iterations include:

- invented logos;
- miscentered title regions;
- inconsistent column grids;
- excess density;
- obsolete product names;
- medical-specific result sections;
- system-platform icon sections that were later removed;
- wording that should not be reused.

Use the documents in `docs/` as the source of truth.
